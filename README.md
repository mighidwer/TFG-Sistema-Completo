# TFG-Sistema-Completo

## Trabajo Fin de Grado

**Optimización de rutas de operadores en sistemas logísticos**

### Descripción

Este repositorio contiene el código fuente completo desarrollado para el Trabajo Fin de Grado, así como todos los recursos necesarios para la ejecución del sistema.

Se incluyen los módulos implementados en Python, los scripts de gestión del servidor OSRM y las matrices de datos previamente generadas para el conjunto de tiendas utilizado en el proyecto. Asimismo, el servidor OSRM se proporciona ya generado, permitiendo la ejecución del sistema sin necesidad de realizar nuevamente el proceso de preprocesado.

### Contenido

* Código fuente completo de la aplicación.
* Scripts de gestión del servidor OSRM.
* Recursos necesarios para el funcionamiento del sistema.
* Matrices de datos generadas.
* Servidor OSRM preparado para su utilización.
