# Carpeta Incidencias

Esta carpeta está destinada al almacenamiento de los archivos generados automáticamente durante la ejecución del sistema relacionados con la detección de incidencias o soluciones no factibles del modelo de optimización.

En concreto, en este directorio se almacenan los ficheros en formato .ilp que contienen la información correspondiente a los modelos que no han resultado factibles durante el proceso de resolución.

En la versión distribuida del proyecto, esta carpeta se encuentra inicialmente vacía, ya que dichos archivos se generan de forma automática durante la ejecución del sistema cuando se detectan situaciones de inviabilidad en los modelos de optimización.
